# Java Beginner Projects

A collection of beginner Java console applications built while learning Java fundamentals.

## Projects

### Mad Libs Game
A console-based Java game where the user enters different words to create a funny story.

**Concepts used:**
- Scanner
- User Input
- String Manipulation
- Console Output

---

### 🛒 Shopping Cart
A simple Java console application that calculates the total price of purchased items.

**Concepts used:**
- Scanner
- Variables
- Arithmetic Operations
- Console Output
